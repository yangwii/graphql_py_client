# GraphQL Client


## How to Use ?

1. start webserver

    > **git clone**

	> **python manager.py start**

2. start webserver

    > **pip install git+https://github.com/yangwii/graphql_py_client.git@master**

	> **graphqlpy_client start**

Then visit [127.0.0.1:5001/index](http://127.0.0.1:5001/index) OK!
